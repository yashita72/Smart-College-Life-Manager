
# Smart College Life Manager

Managing college life can get messy with assignments, exams, and events spread across different apps.  
This project is a simple attempt to bring everything into one place.

Smart College Life Manager is a web-based dashboard where students can log in, add their academic tasks, track deadlines, and stay aware of urgent work.

---

## What this project does
- Allows students to create an account and log in securely
- Lets users add academic tasks like assignments, exams, and events
- Each task includes a deadline and priority
- Automatically highlights urgent and overdue tasks
- Shows reminders when deadlines are close (while the app is active)
- Provides a clean and distraction-free dark UI

---

## Why this project
As students, we often miss deadlines because information is scattered across WhatsApp groups, notes, and calendars.  
This project focuses on solving that problem in a simple and practical way.

---

## Technologies used
- HTML, CSS, JavaScript
- Firebase Authentication (Email & Password)
- Browser Local Storage

---

## Current limitations
- Tasks are stored locally in the browser
- Reminders work only when the application is open

These limitations are intentional to keep the project lightweight for a mini-project / hackathon MVP.

---

## Future scope
- Storing tasks in Firebase Firestore
- Full Google Calendar API integration
- Automated reminders using Firebase Cloud Functions
- Mobile-friendly version
